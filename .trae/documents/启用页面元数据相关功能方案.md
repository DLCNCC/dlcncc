# 启用页面元数据相关功能方案

## 1. 功能分析

根据主题文档 `https://theme-hope.vuejs.press/zh/guide/feature/meta.html`，页面元数据相关功能包括：

- **基于Git的信息**：页面创建时间、最后更新时间、贡献者列表
- **编辑此页链接**：自动生成指向GitHub仓库的编辑链接
- **上一页/下一页导航**：基于侧边栏配置自动生成

## 2. 当前配置状态

当前主题配置已启用：
- `contributors: true`：显示贡献者
- `changelog: true`：显示变更日志

但缺少：
- Git插件启用配置：`plugins.git: true`
- 完整的编辑链接配置

## 3. 配置方案

### 3.1 启用Git插件

在主题配置中添加Git插件配置，用于获取页面Git信息：

```typescript
export default hopeTheme({
  // ... 现有配置
  
  plugins: {
    // ... 现有插件配置
    
    // 启用Git插件获取页面元数据
    git: true,
  },
});
```

### 3.2 配置编辑链接

确保编辑链接相关配置完整（已部分配置）：

```typescript
export default hopeTheme({
  // ... 现有配置
  
  // 已配置的编辑链接基础信息
  repo: "DLCNCC/dlcncc",
  docsDir: "src",
  
  // 可选：如果仓库分支不是main，需要配置
  // docsBranch: "main",
  
  // 可选：全局启用/禁用编辑链接
  // editLink: true,
});
```

### 3.3 配置开发模式下的热重载（可选）

如果需要在开发模式下查看元数据变化，可以启用热重载：

```typescript
export default hopeTheme({
  // ... 现有配置
  
  // 启用热重载（开发模式下实时更新Git信息）
  hotReload: true,
});
```

## 4. 详细实施步骤

1. **编辑文件**：`src/.vuepress/theme.ts`
2. **在plugins对象中添加git插件配置**：`git: true`
3. **检查并完善编辑链接配置**：确保repo、docsDir正确
4. **可选：启用hotReload**
5. **保存文件**
6. **验证**：运行开发服务器查看效果

## 5. 预期效果

- 页面底部显示最后更新时间
- 页面底部显示贡献者列表
- 页面底部显示"编辑此页"链接
- 基于Git提交记录自动生成变更日志

## 6. 注意事项

- Git插件需要调用Git程序，首次启用可能需要初始化
- 开发模式下启用Git插件会影响性能
- 页面元数据基于Git提交记录，需确保文件已提交到Git仓库

通过以上配置，即可完整启用主题的页面元数据相关功能。
